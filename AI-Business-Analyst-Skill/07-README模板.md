## 使用说明

1. 将模板复制到新项目的 `README.md`
2. 用 `[ ]` 标记的占位符替换为实际内容
3. 确保第一句话是 CEO 能理解的核心结论
4. 技术细节只放在"附录"章节
5. 每个表格至少有一个"关键判断"或"核心洞察"标注
