## 📊 核心方法论

### 三级分析架构

```
Layer 1: 描述性 (What happened?)
    └── 12 维度 EDA — 全面了解经营全貌
    
Layer 2: 诊断性 (Why did it happen?)  
    └── 8 维度交叉下钻 — 精确定位问题根因
    
Layer 3: 规范性 (What should we do?)
    └── 3-5 策略 + ROI 模拟 + 实施路线图
```

### AI 原生工作流

```
Prompt（中文）
    ↓
AI 分析设计（框架 + 代码骨架）
    ↓
Python 执行（数据聚合 + 统计检验）
    ↓
人工校验（合理性 + 业务逻辑）
    ↓
BI 呈现（Plotly → HTML 仪表板 → 咨询报告）
```
